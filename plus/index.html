    # 插件

    安装插件 vuepress-plugin-autobar (opens new window)

        npm install -D boboidream/vuepress-bar
    
    // 配置中添加插件  .vuepress/config.js
    odule.exports = {
      plugins: ['autobar']
    }

    # Tips

    1. 拼音地址兼容 vuepress-plugin-permalink-pinyin (opens new window)
        npm i vuepress-plugin-permalink-pinyin --save-dev
    plugins: ['permalink-pinyin']
     
    // 配置中添加参数 .vuepress/config.js
    module.exports = {
      plugins: ['permalink-pinyin', ['autobar', {'pinyinNav': true}]],
    }
    1. 如果觉得生成链接太丑，可以使用 vuepress-plugin-rpurl (opens new window) 插件，此插件会默认将 autobar 生成链接优化
        npm install boboidream/vuepress-plugin-rpurl -D
    
    plugins: ['rpurl']
